INSTALACIÓN (solo la primera vez)
----------------------------------
1. Copia esta carpeta donde quieras (temporal)
2. Doble clic en: INSTALAR.bat
3. El instalador:
   ✓ Crea D:\BancoComunal\ automáticamente
   ✓ Copia todos los archivos allí
   ✓ Instala las dependencias de Python (reportlab, bcrypt, google-generativeai, python-docx, python-pptx, faker)
   ✓ Crea icono 🌱 en el Escritorio
   ✓ Abre la app automáticamente

4. Completa la configuración inicial:
   - Nombre de tu Comuna y Banco
   - Estado y Municipio (para centrar el mapa)
   - Datos de las Voceras/Voceros
   - Tu usuario y contraseña
   - API Key de Google Gemini (para el OCR inteligente)

USO DIARIO
-----------
Doble clic en el ícono 🌱 SemilleroComunal del Escritorio
¡Un solo clic, sin ventanas de comandos!

RECUPERAR CONTRASEÑA
---------------------
Si olvidas tu contraseña:
1. Abre http://localhost:5000/recuperar
2. Responde tu pregunta de seguridad
3. Crea nueva contraseña
4. Tus datos NO se pierden
